## Docker
```bash
$ docker-compose up
```

## Instalação manual

Não esqueça de alterar o POSTGRES_HOST que está no arquivo .env!

```bash
$ npm install
```

## Executando aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Teste

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
